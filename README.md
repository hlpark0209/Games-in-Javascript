## 📑 Functional List
* '+' 버튼 or 키보드 enter key 클릭으로 동작
* 등록한 요소가 리스트에 append하는 기능 
* 삭제 아이콘 ( 휴지통 ) 클릭 시, 요소 삭제
* 완료한 요소는 취소선으로 처리
* list 요소가 지정 높이보다 초과될 시, 스크롤 생성
* list 요소 지속적으로 추가될 시, 해당 요소의 위치로 이동 (scrollview)
